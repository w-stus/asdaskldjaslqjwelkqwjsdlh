Частина 1 — JavaScript Tabs

Вкладки (Таби)

Загальний опис

Цей продукт — результат виконання лабораторної роботи №15 з курсу Веб-технологій. Сторінка демонструє три ключові техніки: систему вкладок (табів) для організації контенту, модальні вікна для інтерактивних форм та публікацію готового сайту на GitHub Pages.

Таби реалізовані на чистому JavaScript без бібліотек. При кліку JS прибирає клас .active з усіх елементів і додає його поточній кнопці та відповідному блоку контенту. Анімація появи — CSS @keyframes fadeIn.

Головна перевага табів — розміщення великого обсягу інформації в одному місці без прокрутки. Користувач самостійно обирає, який розділ переглядати.

Доставка та підтримка

Сайт розгорнуто на GitHub Pages — безкоштовному хостингу для статичних проєктів. Жодних серверів, баз даних чи щомісячної оплати.

Головний файл обов'язково повинен називатись index.html (маленькими літерами) — саме його GitHub Pages завантажує першим при переході за посиланням.

Оновлення сайту: завантажте нові файли у репозиторій через "Upload files" або за допомогою Git. Зміни з'являться на сайті через 1–2 хвилини.

Частина 3 — Публікація

GitHub Pages

01

Реєстрація

Створіть акаунт на github.com, якщо ще не маєте. Ваш username буде частиною URL сайту.

02

Репозиторій

New Repository → назва → оберіть Public → Create. Приватні репозиторії не публікуються безкоштовно.

03

Завантаження

"Upload files" → перетягніть index.html, папки css/, js/, img/ → напишіть commit message → Commit changes.

04

Settings → Pages

Settings → Pages → Branch: main → Save. Зачекайте 1–2 хв та оновіть сторінку.

05

Готово!

Посилання у форматі username.github.io/repo-name з'явиться зверху на вкладці Pages.

06

Оновлення

Завантажте нові файли у репозиторій. Зміни відобразяться на сайті автоматично за 1–2 хвилини.

Важливо: головний файл сайту обов'язково повинен називатися index.html (маленькими літерами). Саме цей файл GitHub Pages завантажує першим при переході за посиланням. Якщо файл називається Index.html або index.HTML — сайт не запуститься.
